tinysparql3-import(1)
=====================

== NAME

tinysparql3-import - Import data into a TinySPARQL database.

== SYNOPSIS

*tinysparql3 import* FILE.ttl

== DESCRIPTION

*tinysparql3 import* imports data into a TinySPARQL database.

The data must conform to the existing ontology of the database.

The data must be in Turtle format. You can use a tool such as rapper(1)
to convert the data from other formats to Turtle.

== SEE ALSO

*tinysparql3-export*(1), *tinysparql3-sparql*(1).
